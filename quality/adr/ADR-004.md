## ADR-004: CLAUDE.md managed policy (prompt-level enforcement)
**Status:** Accepted
**Context:** Technical measures cannot prevent Claude from following instructions the user gives it. A policy file loaded at highest precedence instructs Claude about what it should not do (read credentials, send data externally, etc.).
**Decision:** `CLAUDE.md` is baked into the image at `/etc/claude-code/CLAUDE.md` (read-only, chmod 444). Claude Code loads it at system level — overrides project and user instructions.
**Consequences:**
+ Raises bar for accidental misuse significantly
+ Clear audit artifact: "the agent was instructed not to exfiltrate data"
- Prompt-level only — a sufficiently adversarial prompt could override (acknowledged in README)
- No technical enforcement for clipboard or mounted volumes
